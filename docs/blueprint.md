# **App Name**: MA Discovery

## Core Features:

- Experiência de Navegação Interativa: Navegação fluida entre todas as telas (Abertura, Seleção de Jogo, Mini Jogo, Resultado, Avaliação, CTA) com design responsivo e microinterações, otimizada para toque.
- Mini-Game 'Caixa Premiada': Implementação do jogo principal 'Caixa Premiada' com animação de caixas e revelação de resultado em estilo premium.
- Geração e Exibição de Cupons: Geração automática de códigos de cupom exclusivos no formato 'MA-XXXX' para vantagens desbloqueadas e exibição ao usuário.
- Linkagem para Avaliação e Canais de Venda: Funcionalidade para abrir diretamente o link de avaliação da loja no Google Maps e redirecionar para WhatsApp ou ofertas da loja.
- Armazenamento de Dados de Sessão e Cupons: Persistência no banco de dados para registrar sessões de jogo, cupons gerados, data, resultado e status de uso do cupom.
- Painel de Controle Administrativo Simplificado: Dashboard para visualização das métricas principais: número de partidas, cupons gerados/usados e cliques no link de avaliação, com um layout minimalista.
- Acesso via QR Code Móvel: Gerar um QR Code na tela final para acesso rápido a ofertas, permitindo a experiência também em dispositivos móveis dos clientes.

## Style Guidelines:

- Esquema de cores dark, com Primary Color (para elementos interativos e brilhos): um azul vibrante (#2E97F5), evocando tecnologia e modernidade. Background Color (principal): o 'azul marinho profundo' da marca (#0B1F3B), com uma saturação controlada para ser a base elegante e escura da interface. Accent Color (para destaque): o 'dourado metálico' da marca (#C9A24A), utilizado em detalhes importantes para reforçar a percepção de luxo e prêmio.
- Todo o texto da interface utilizará a fonte 'Space Grotesk', uma sans-serif com toque tecnológico e legível, alinhada à estética clean e moderna de tecnologia premium.
- Utilização de ícones minimalistas e monocromáticos, seguindo a estética clean e de fácil reconhecimento, com um estilo similar ao System UI de sistemas operacionais premium, como o iOS.
- Layout clean e minimalista, priorizando espaços em branco (ou no caso, espaços em 'dark') para uma apresentação de conteúdo premium. Elementos flutuantes, como o iPhone 3D, centralizados para atrair a atenção. Utilização sutil de glassmorphism e desfoque para profundidade e elegância, com reflexos suaves e sombras macias em elementos-chave.
- Microinterações com animações fluidas e tempos de duração entre 200ms e 400ms. Partículas leves de fundo para uma ambientação sutil. Efeitos de glow azul em elementos interativos e animações dedicadas, como a abertura elegante da 'Caixa Premiada'. Feedback tátil com vibrações sutis para interação no touch, se suportado.
